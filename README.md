# 🍽️ EatFit Blog

A responsive food blog website built using React and Vite. It showcases healthy recipes, articles, and food-related content with a modern user interface.

## 🚀 Features

- Responsive design
- React components
- Modern UI
- Fast loading with Vite
- Clean and reusable code

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript
- HTML5
- CSS3

## 📂 Project Structure

```
src/
 ├── assets/
 ├── Project/
 │    ├── Navbar/
 │    ├── Title/
 │    ├── Cards/
 │    ├── Footer/
 │    └── Subnavbar/
 ├── App.jsx
 └── main.jsx
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Sushmavj14/EatFit-Blog.git
```

Go to the project folder:

```bash
cd EatFit-Blog
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
